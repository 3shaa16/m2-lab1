# JavaScript Basics Lab

A simple JavaScript project that introduces fundamental concepts through small interactive exercises, including a **Rock, Paper, Scissors** game.

## 📌 Features

* Uses prompts and alerts for user interaction
* Demonstrates variables, conditionals, and basic logic
* Includes a Rock, Paper, Scissors game against the computer
* Random computer choices using JavaScript

## 🛠️ Technologies

* HTML5
* JavaScript

## 🚀 How to Run

1. Open `index.html` in a browser
2. Follow the prompts to play the game and see results

## 📂 Files

* `index.html` – main file
* `script.js` – contains game logic and exercises
